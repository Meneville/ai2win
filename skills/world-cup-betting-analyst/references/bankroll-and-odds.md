# 资金管理与赔率参考

## 默认资金池

用户未指定时，总财富/资金池为 4000。若用户之后提供实际投注、盈利或亏损，用：

`新资金池 = 上次资金池 - 已下注未结算金额 + 已结算返还金额`

记录时区、比赛、玩法、下注金额、赔率、赛果、返还、净利润。

## 仓位框架

保守策略：

- 单注 1%-2% 为常规仓位；强价值且信息稳定时最高约 3%。
- 三场总暴露通常控制在 8%-12% 以内。
- 高方差玩法（比分、进球者、半全场、串关）控制在单注 0.25%-1%。

激进策略：

- 单注 3%-5% 为常规上限；只有高置信正期望才接近 7%。
- 三场总暴露通常不超过 20%-25%。
- 串关只作为小额增强，不得成为主要仓位。

禁止：

- 马丁格尔、追损、用亏损作为加仓理由。
- 因“必胜”“稳胆”突破暴露上限。
- 忽略赔率下跌后的价值消失。

## 概率与期望值

十进制赔率的盈亏平衡概率：

`break_even = 1 / decimal_odds`

单注期望值：

`EV = model_probability * decimal_odds - 1`

只有当 EV 为正且信息质量足够时才建议下注。小幅正 EV 但阵容不确定、赔率快速下滑或盘口高度相关时，可以建议观望或降低仓位。

## 去除水钱

1X2 隐含概率：

`p_home_raw = 1 / odds_home`

`p_draw_raw = 1 / odds_draw`

`p_away_raw = 1 / odds_away`

博彩公司总水钱：

`overround = p_home_raw + p_draw_raw + p_away_raw`

归一化概率：

`p_home = p_home_raw / overround`

`p_draw = p_draw_raw / overround`

`p_away = p_away_raw / overround`

## 水位变化解读

记录：

- 初盘、即时盘、变化方向、变化幅度、更新时间。
- 多家同时降赔通常比单家变化更有参考价值。
- 热门队伍降赔可能来自大众投注，不一定代表真实胜率上升。
- 临近开赛的阵容消息导致盘口变化，权重大于早盘噪音。

## Kelly 参考

可用小分数 Kelly 作为辅助，不要机械套用：

`kelly_fraction = (decimal_odds * p - 1) / (decimal_odds - 1)`

建议最多使用 1/4 Kelly，并受保守/激进总暴露上限约束。若模型概率置信度低，将 Kelly 结果再打折。
